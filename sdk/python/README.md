# Veridex Python SDK and CLI

The Veridex Python SDK allows you to easily interact with the Veridex Agent platform programmatically.

It also includes the `veridex` CLI tool for terminal-based interactions.

## Installation

```bash
pip install veridex
```

## CLI Usage

```bash
veridex login --token YOUR_TOKEN
veridex agent run --goal "Analyze Q3 metrics"
```
